# AI Fundamentals
## 1. Tokens - How AI reads your code
- Word pieces, sometimes 1 word,part-word, punctuation, whitepsace or special char
- Longer prompt = more tokens = higher cost
- Code uses more tokens than prose

## 2. Context Window - AI's short-term memory
- Measured in tokens

## 3. Hallucinations 
- When AI confidently makes things up 
- Why: It just predicts likely tokens

## 4. Prompt
- what, how, return type

## 5. AI Usage
AI is excellent at "how", you decide "what" and "why"

# Getting started with Github Copilot in VSCode
## 1. Installation and Setup
1. Install Github Copilot Chat Extension
2. Bottom-left, Accounts, Login to Github
3.  Github Copilot Chat Extension > Account preference, verify you can see your Github username 

## 2. Tutorial #1: Code generation via comments
1. Enter comments
2. Copilot generates code
3. Tab to accept all, Enter to accept line

## 3. Github Copilot Modes
1. Ask: Questions and explanations
2. Edit: Modify existing code
3. Agent: Autonomous development
